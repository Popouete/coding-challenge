Auteur: Antony Veilleux
Title: Aidez Papi Ninja!

But: aider Papi ninja à retrouver son nom de ninja!

-Versionnage: 
GitHub : https://github.com/Popouete/coding-challenge

-Hébergement: 
Heroku : https://tlm-code-challenge.herokuapp.com/ninjify

*Première expérience avec Expressjs.*

3 views: 
-ninjify (index)
-ninjify-name (resultat de la recherche)
-404 (error)

App:
Entrer le nom d'un langage de programmation dans l'input.
En cliquant sur le bouton, l'app cherchera ce langage dans le fichier "langage.json"
Si la recherche est positive, on affichera le résultat de la recherche en plus 
d'afficher un 2 autre valeur aléatoire :
-un titre ex: Maitre, Ninja etc.
-un nom de famille : Tao, WongTong etc.

Exemple de résultat: 
Caporal PHP Lee , 
General C++ Tao , 
Maitre Python WongTong

***ATTENTION: il est possible de trouver la totalité des langages de programmation dans cet app***
#Konami #NinjyfyBottomPage #Onclick()

Have fun!
